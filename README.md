### Getting Started

```
npm i
```

```
npm start
```

This is should install all the dependencies and fire up **localhost:3000**.

## Features

- View All Posts
- View All Users
- View Posts per User
- Add, Delete, Update Posts

_No Authentication/Data Persistence Yet!_
